# 个性化的学习推荐和促销

您可以进行选择，以允许我们根据您在 Adobe 产品中的工作，为您提供量身定制的有用内容。从以下选项中选择您的偏好设置，系统将根据下列条件向您提供相关的学习信息：

* 您的技能水平和角色
* 您的工作流
* 您可能会使用的功能
* 等等！

[自定义学习](https://experience.adobe.com/?shell_forceuserconsent=true#/home)

您可以随时在[用户偏好设置](https://experience.adobe.com/preferences/)中更新您的选择。

| 面向您的 Adobe 产品和服务的个性化培训和支持推荐 |  |  |
| 通过任意这三种方式，根据您的 Adobe 产品使用情况，获取相关的最佳实践、提示和技巧、有用的演练等... |  |  |
|--- |--- |--- |
| 在您的 Adobe 产品中... | 查看弹出窗口和工具提示以获取实时帮助。 | 这可能包括... <ul><li>Adobe 专家提供的分步指南和有用提示</li> <li>视频教程和信息性演练</li> <li>深度培训和教育内容</li> <li>视频和文章推荐</li></ul> |
| 在 Adobe 发给您的电子邮件中... | 查看与您在产品中的工作相关的学习内容。 | ^ |
| 在 Experience League 社区中... | 根据您当前的工作，查看个性化的帖子和文章推荐。 | ^ |
