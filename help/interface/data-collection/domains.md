# Experience Cloud 域 {#domains}

Experience Cloud使用以下主机来交付应用程序、提高性能和改善产品体验。 Adobe建议允许这些域通过贵组织的网络或防火墙以获得最佳体验。 其他Experience Cloud产品和服务使用自己的域 — 有关更多信息，请查看这些应用程序的文档。

* Adobe Analytics
* Customer Journey Analytics

| 技术 | 域 |
| --- | --- |
| Adobe Experience Cloud 域 | `adobe.com`、`adobe.net`、`adobe.io` |
| Adobe Identity Management Service (IMS) | `adobelogin.com` |
| Experience Cloud 字体 | `typekit.net` |
| 数据收集（针对产品性能和使用情况） | `adobedtm.com` |
| Gainsight（用于产品指导和帮助） | `esp.aptrinsic.com` |
