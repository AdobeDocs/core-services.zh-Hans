# Experience Cloud 集成

Adobe Experience Cloud 是一套业内最佳的综合性集成解决方案，它基于常用的数据平台而构建，提供了一组功能强大的通用核心服务。

## 核心服务 - 为核心服务启用解决方案 {#section_A3D024994DA3492F8435CFCC4EF035C2}

帮助：[为核心服务启用解决方案](core-services/core-services.md#concept_07ED1D5C64234E77976E6D572E78FB9C)

描述如何执行以下操作：

* 在 Experience Cloud 中配置您的公司。
* 使您成为管理员。
* [实施 Experience Cloud ID 服务](https://marketing.adobe.com/resources/help/en_US/mcvid/)。
* 通过 DTM 使您的 Analytics 和 Target 实施符合现代化要求。
* 开始使用核心服务.

解决方案或服务：

* 激活 - Dynamic Tag Management 或 Experience Platform Lauch
* Analytics
* Target
* [Experience Cloud ID 服务](https://marketing.adobe.com/resources/help/en_US/mcvid/)

## Experience Cloud ID 服务 {#section_6ECCCFA2D84D4D4F88C879C799CA9D78}

Help: [Experience Cloud ID Service](https://marketing.adobe.com/resources/help/en_US/mcvid/)

ID 服务提供了一个通用的持久 ID，可用来在 Experience Cloud 的所有解决方案中标识您的访客。它可以取代如下服务的 ID 生成代码：Analytics、Audience Manager、Target、视频心跳以及其他 Experience Cloud 解决方案或功能。

**适用的解决方案或服务**

* [Analytics](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-analytics.html)
* [Target](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-setup-target.html)
* [Data Workbench](https://marketing.adobe.com/resources/help/en_US/mcvid/mcvid-dwb.html)

## 受众{#section_5F60D7B0833348B9A1D74663AADCB42C}

帮助：[受众](audience-library/audience-library.md#topic_679810123CAA4E0CA4FA3417FB0100C7)

在 Experience Cloud 受众库中创建和管理受众。受众可从各种来源创建或派生，如：

* 在 [!DNL Experience Cloud] 中创建新受众。
* 从 [!DNL Analytics] 发布到 [!DNL Experience Cloud] 的区段。
* 从 [!DNL Audience Manager].

**适用的解决方案或服务**

* Adobe Target 中的[活动](https://marketing.adobe.com/resources/help/en_US/target/target/c_activities.html)
* Audience Manager 中的[分段](https://marketing.adobe.com/resources/help/en_US/analytics/audiences/aam-analytics-segments.html)
* [Media Optimizer](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm)（需要登录）

## 客户属性 {#section_6A9EA6847F654F129381869E5016626C}

帮助：[客户属性](attributes/attributes.md#concept_ACFEE7C8B8E94875BA0825CDF4913AF1)

如果您在客户关系管理 (CRM) 数据库中捕获到企业客户数据，则可以将该数据上传到 Experience Cloud 中的客户属性数据源。上传后，即可利用 [!DNL Adobe Analytics] 和 [!DNL Adobe Target] 中的数据。

**适用的解决方案或服务**

* Analytics：[客户属性报表](https://marketing.adobe.com/resources/help/en_US/reference/reports_customer_attributes.html)
* Target：配置 Target 的客户属性[订阅](https://marketing.adobe.com/resources/help/en_US/mcloud/subscription.html)

## 激活 {#section_A23510A2D57842F6BAD043650C06DE42}

使用 Adobe Experience Cloud 调试器扩展来调试 Auditor 发现的问题或检查有关您实施的其他信息。

**适用的解决方案或服务**

## Advertising Cloud {#section_9B1935F8BBC147C89C6DB68A35CB1BAB}

Help (login required): [Integration with Adobe Experience Cloud Solutions and Core Services](https://enterprise.efrontier.com/CMDashboard/help/internal/concepts_and_features/media_optimizer_integration_with_adobe_marketing_cloud.htm)

**适用的解决方案或服务**

**Analytics：**&#x200B;可以每天将网站参与度和转化数据发送到 Media Optimizer，以便将这些数据用于广告优化和报告。此外，Media Optimizer 也可以每天将搜索引擎和社交网络流量数据发送到 Analytics，以便将这些数据用于 Reports &amp; Analytics、Report Builder 和 Ad Hoc Analysis 功能中的报表。

**动态标签管理器：**&#x200B;您可以[使用动态标签管理器创建 Media Optimizer 基于像素的转化跟踪标签](https://marketing.adobe.com/resources/help/en_US/dtm/t_media_opt_rule.html)及第三方跟踪标签，以便用于搜索、社交和显示广告登录页面。（您也可以直接在 Media Optimizer 中创建 Media Optimizer 标签。）

**Experience Cloud 受众：**（具有显示管理需求的广告商）您可以将任何 [Adobe Experience Cloud 受众](https://marketing.adobe.com/resources/help/en_US/mcloud/audience_library.html)用作显示广告的目标。您可以自动使用在 Adobe Experience Cloud 中创建的受众及 Analytics 中已发布到 Adobe Experience Cloud 的受众，也可以使用来自 Audience Manager 的受众，前提是已将 Media Optimizer 帐户配置为允许这样做。请联系帐户管理员以了解有关访问 Adobe Experience Cloud 及其 Profiles &amp; Audiences 核心服务的详细信息，以及有关 Media Optimizer 与 Adobe Experience Cloud 受众之间初始设置的详细信息。**注意：**&#x200B;如果您还使用 Adobe Target，则已发布到 Adobe Experience Cloud 的任何受众也可用于 Target 中的各个活动。

**Experience Cloud 资产：**（具有显示管理需求的广告商）您可以通过新的显示测试版视图，将任何 Adobe Experience Cloud 资产用作显示广告的创意。必须[通过 Adobe Experience Cloud 登录到 Media Optimizer](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm)，才能访问您的 Adobe Experience Cloud 资产。有关访问 Adobe Experience Cloud 的信息，请联系您的帐户管理员。

**Experience Cloud 通知：**&#x200B;通过每个页面顶部的通知链接，可以查看从测试版搜索警告模板生成的所有警告，以及 Adobe Experience Cloud 系统更新、帖子、提及次数和共享资产。必须[通过 Adobe Experience Cloud 登录到 Media Optimizer](https://enterprise-test.efrontier.com/CMDashboard/help/internal/getting_started/t_log_in_from_adobe_marketing_cloud.htm)，才能访问您的通知。有关访问 Adobe Experience Cloud 的信息，请联系您的帐户管理员。
